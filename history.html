Key Milestones in Version Control History

The most important systems and events in the evolution of version control:

  1. 1972 — SCCS (Source Code Control System): the beginning of source control.
  2. 1982 — RCS (Revision Control System): introduced delta storage for file changes.
  3. 1986 — CVS: enabled collaboration using a client–server model.
  4. 2000 — Subversion (SVN): improved CVS with atomic commits and better branching.
  5. 2005 — Git and Mercurial: the rise of distributed version control systems (DVCS).
  6. 2008 — GitHub: brought social collaboration, pull requests, and shared development.
